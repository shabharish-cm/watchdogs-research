## 🗓️ Initial Seed Run — Last 60 Days (2026-01-15 to 2026-03-17)

**Top 10 Problems from Product Hunt launches (Jan–Mar 2026):**

### 1. Manual Content Creation Slowness — Score: 70.4
- **Prevalence**: 8/10 — Impacts writers, marketers, designers, and content teams across industries
- **Build Ease**: 8/10 — Leveraging existing LLMs and design APIs make this feasible
- **Example products solving this**: Migma (email templates), TravelAnimator (map animations), ChartGen AI (data viz), Stitch by Google (UI from sketches)
- **Solo Dev Angle**: Build a niche content generator (emails, social captions, product descriptions) powered by Claude/GPT with smart templates.

### 2. Repetitive Workflow Automation — Score: 69.3
- **Prevalence**: 9/10 — Every knowledge worker faces repetitive tasks (data entry, form filling, report generation)
- **Build Ease**: 7/10 — Requires LLM integration, possibly workflow orchestration, but doable solo
- **Example products solving this**: Aident AI, Instruct, Anything API
- **Solo Dev Angle**: Start with one vertical (e.g., HR onboarding, expense reporting). Build an agent that understands intent in plain English.

### 3. Manual Email Template Creation — Score: 53.9
- **Prevalence**: 7/10 — Every business sends marketing, transactional, or internal emails
- **Build Ease**: 7/10 — Use modern email frameworks (MJML), AI code gen, and template libraries
- **Example products solving this**: Migma
- **Solo Dev Angle**: AI email builder with drag-and-drop + code gen hybrid. Target agencies or SaaS founders first.

### 4. Note Organization With AI Search — Score: 53.9
- **Prevalence**: 7/10 — Knowledge workers, students, and researchers accumulate notes daily
- **Build Ease**: 7/10 — Local-first storage, vector search, and Claude integration feasible
- **Example products solving this**: Kuku
- **Solo Dev Angle**: Lightweight note app with semantic search + AI tagging. Build for a niche (engineers, writers).

### 5. Coding Error Fix Iteration Cycle — Score: 46.2
- **Prevalence**: 7/10 — Every developer experiences the write-test-debug-fix loop
- **Build Ease**: 6/10 — Requires code parsing, execution feedback loops, and IDE integration
- **Example products solving this**: NOVA
- **Solo Dev Angle**: Focus on one language. Build a CLI tool that auto-fixes errors in test output.

### 6. UX/Design Review & Iteration — Score: 46.2
- **Prevalence**: 7/10 — Product teams spend hours on design feedback cycles
- **Build Ease**: 6/10 — Needs Figma/design tool integration, vision AI, and UX heuristics
- **Example products solving this**: Figr AI
- **Solo Dev Angle**: Lightweight Figma plugin that flags common UX issues (contrast, spacing, mobile).

### 7. Sales Outreach Personalization — Score: 46.2
- **Prevalence**: 7/10 — B2B sales teams need to reach hundreds/thousands per month
- **Build Ease**: 6/10 — Requires prospect data API, email sending, LLM for personalization
- **Example products solving this**: Starnus
- **Solo Dev Angle**: Build for a niche (e.g., SaaS founders → agencies). Integrate with a free email provider.

### 8. Complex Data Into Visuals — Score: 44.0
- **Prevalence**: 8/10 — Analysts, PMs, and executives need charts and dashboards daily
- **Build Ease**: 5/10 — Requires charting library expertise, data parsing, and design sense
- **Example products solving this**: ChartGen AI
- **Solo Dev Angle**: Specialize in one data format (CSV, JSON) or domain (ecom metrics, analytics).

### 9. Code Quality & Bug Detection Lag — Score: 44.0
- **Prevalence**: 8/10 — Teams want to catch bugs early and maintain standards
- **Build Ease**: 5/10 — Requires AST parsing, linting, and integration with CI/CD
- **Example products solving this**: Enia Code
- **Solo Dev Angle**: Build a GitHub Action or IDE extension that flags architectural debt.

### 10. Accessing Non-API Website Data — Score: 38.5
- **Prevalence**: 7/10 — Many business workflows depend on scraping data from websites without APIs
- **Build Ease**: 5/10 — Browser automation, Playwright/Selenium, LLM instruction parsing needed
- **Example products solving this**: Anything API
- **Solo Dev Angle**: Build a simpler version for one website type (job boards, e-commerce, real estate).
